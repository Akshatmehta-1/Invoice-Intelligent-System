# 🧠 Vendor Invoice Intelligence Portal

AI-powered system to **predict freight cost** and **flag risky invoices** using machine learning.

---

## 🚀 What This Project Does

* Predicts **freight cost** from invoice amount
* Flags **suspicious invoices** (safe vs risky)
* Reduces manual review effort and improves decision-making

---

## 🧩 Key Features

* 📊 Freight Cost Prediction (Regression model)
* 🚩 Invoice Risk Detection (Classification model)
* ⚡ Real-time predictions via Streamlit UI
* 🧠 Random Forest models
